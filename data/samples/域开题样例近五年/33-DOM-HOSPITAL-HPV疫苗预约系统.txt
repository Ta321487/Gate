本科毕业设计（论文）开题报告

题目：基于 Spring Boot 与 Vue 的 HPV 疫苗预约管理系统的设计与实现

开题年份：2025

一、选题背景与意义

1.1 选题背景
适龄人群对 HPV 等疫苗接种的预约需求上升。社区门诊或校医院在放号后，常出现电话占线、现场排队领号、预约信息登记不清等情况。部分接种点仍用表格登记预约人、剂次与意向时段，改约与爽约后号源回收不及时，工作人员核对困难。

建设疫苗接种预约管理系统，可按接种门诊或疫苗种类发布可约时段，预约人在线提交预约，接种点完成接种办结或处理取消，从而提升预约组织效率。本课题以预约占号与办结为主，不扩展疾控直报与冷链物联网。

1.2 选题意义
（1）方便群众按时段预约疫苗接种；
（2）减少窗口电话与现场登记负担；
（3）形成可查询的预约与接种办结记录；
（4）完成公共卫生场景下预约类系统的设计实现。

二、国内外研究现状

国外免疫规划信息系统较成熟；国内各地接种平台能力不一，商业系统常绑定区域项目。本科毕设实现通用 Web 预约演示即可，数据采用模拟信息并注意隐私表述。

三、研究目标与主要内容

3.1 研究目标
设计并实现基于 Spring Boot 与 Vue 的 HPV 疫苗预约管理系统，支持预约用户与接种点管理员，完成疫苗/门诊资源浏览、分时预约、取消与接种办结、公告通知。

3.2 主要功能
1. 用户登录与个人资料维护；
2. 疫苗种类或接种门诊、可约时段浏览；
3. 在线提交疫苗接种预约；
4. 预约取消与接种办结；
5. 本人预约记录查询；
6. 接种须知与放号公告管理；
7. 管理端预约统计概览。

3.3 拟解决关键问题
（1）分时号源占用，防止超额预约；
（2）预约、取消、已接种等状态管理；
（3）用户端与管理端权限分离。

四、技术方案

Spring Boot + Spring Security + MyBatis + MySQL；Vue 3 + Element Plus。注意演示数据脱敏说明。

五、非本期范围

疾控系统直报对接、疫苗冷链温控物联网、接种禁忌智能问询量表引擎、真短信通知通道不在本期实现。

六、进度安排

第 1–2 周：调研预约接种流程并开题；
第 3–4 周：需求与库表设计；
第 5–10 周：资源与预约模块开发；
第 11–12 周：测试与演示；
第 13–16 周：论文与答辩。

七、预期成果

可演示系统、数据库脚本、毕业论文与答辩材料。
